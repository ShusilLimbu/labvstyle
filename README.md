# 🛒 E-commerce Web Application (Full Stack)- LabVStyle

A full-stack e-commerce web application built using the MERN stack.  
This project demonstrates backend architecture design, REST API development, authentication, and frontend-backend integration.

---

## 🚀 Tech Stack

- **Frontend:** React.js  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB / SQL  
- **Tools & Libraries:** JWT, Axios, Postman, Git, GitHub  

---

## ✨ Features

- 🔐 User authentication with JWT (Login / Signup)
- 👥 Role-based access control (Admin / User)
- 🛍️ Product listing and management
- 🛒 Shopping cart functionality
- 📦 Order creation and management
- ⚙️ Modular and scalable backend architecture
- 🔄 RESTful API integration between frontend and backend
- 🧪 API testing and debugging using Postman

---

## 🏗️ Project Architecture

- **Frontend (React.js):** Handles UI and user interactions  
- **Backend (Node.js + Express):** Manages APIs and business logic  
- **Database (MongoDB/SQL):** Stores users, products, carts, and orders  

---

## 🔑 Key Highlights

- Secure authentication using JWT
- Optimized database schema design
- Clean separation of concerns (MVC-style structure)
- Efficient API communication using Axios
- Version control with Git & GitHub
